# 프롬포트랩

결과 이미지와 프롬프트 원문을 함께 보고 공유하는 이미지 프롬프트 커뮤니티 프로토타입입니다.

## 주요 기능

- 이미지 프롬프트 갤러리
- 결과 이미지, 모델, 비율, 태그 표시
- 프롬프트 상세 페이지
- 댓글 예시
- 글 중심 게시판
- 실시간 베스트 랭킹
- 로컬 프로토타입 로그인 UI
- SEO용 `robots.txt`, `sitemap.xml`

## 실행

```bash
npm install
npm run dev
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:3000
```

## 배포

Vercel 배포를 기준으로 만들어져 있습니다.

```bash
npm run build
```

빌드가 통과하면 Vercel에서 GitHub 저장소를 연결해 배포할 수 있습니다.
